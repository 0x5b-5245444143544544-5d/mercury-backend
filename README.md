## Features
- [x]  Bulk Mailing
- [x]  Login/Logout
- [x]  MJML to HTML conversion
- [x]  Dynamic content insertion

<br>

## Dependencies
 - Django
 - Django REST framework
 - boto3

<br>

## Running


### Command to install
```bash
pip install -r requirements.txt
```

### Commands to start the development server

```bash
cd mercury
python manage.py runserver
```
